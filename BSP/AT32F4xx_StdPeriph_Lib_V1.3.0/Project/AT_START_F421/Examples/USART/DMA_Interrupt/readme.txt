@par Example Description 

  This demo provides a basic communication between USART1 and USART2 using DMA 
  capability, flags and interrupts.If received data right ,the Three leds will 
  turn on.
  
  - Connect USART1 Tx pin (PA.09) to USART2 Rx pin (PA.03)
  - Connect USART1 Rx pin (PA.10) to USART2 Tx pin (PA.02)
