@par Example Desctiption
  This demo is based on the AT-START-F421 board.
  The Standby mode exit by the Rising edge of WKUPx(x = 1/2/6/7):
	WKUP1！！！！！！PA0
	WKUP2！！！！！！PC13
	WKUP6！！！！！！PB5
	WKUP7！！！！！！PB15
  LED2 lights means exit standby.
  LED3 lights means  standby mode exit by WKUPx Pin's Rising edge.
