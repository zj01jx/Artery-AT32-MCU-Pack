@par Example Description 

  This demo is based on the AT-START-F413 board,in this demo,the code contains three project targets. Please use corresponding target 
  according to your SRAM settings.
  step 1:
  rebuild all target files.
  step 2:
  Select target "set_SRAM_16K", download. If everything is ok, usart1 will print test success message.
  step 3:
  Select target "set_SRAM_32K", download. If everything is ok, usart1 will print test success message.
  step 4:
  Select target "set_SRAM_64K", download. If everything is ok, usart1 will print test success message.
