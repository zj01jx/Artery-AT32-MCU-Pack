@par Example Description 

  This demo provides a basic communication between USART2 and USART3 using 
  interrupts.If received data right ,the Three leds will turn on.
  Set-up 
  - Connect USART2 Tx pin (PA.02) to USART3 Rx pin (PC.11)
  - Connect USART2 Rx pin (PA.03) to USART3 Tx pin (PC.10)
