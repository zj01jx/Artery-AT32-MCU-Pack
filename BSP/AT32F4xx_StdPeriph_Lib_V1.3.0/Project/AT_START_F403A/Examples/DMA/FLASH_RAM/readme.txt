@par Example Description 
 
  @Description
  This demo provides a description of how to use a DMA channel to transfer a 
  word data buffer from FLASH memory to embedded SRAM memory.DMA1 Channel6 is 
  configured to transfer the contents of a 32-word data buffer stored in Flash 
  memory to the reception buffer in RAM.If received data right ,the Three leds 
  will turn on.
