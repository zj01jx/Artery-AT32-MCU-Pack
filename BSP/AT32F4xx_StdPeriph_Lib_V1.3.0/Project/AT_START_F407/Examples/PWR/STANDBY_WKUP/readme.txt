@par Example Desctiption
  This demo is based on the AT-START-F407 board.
  The Standby mode exit by the Rising edge of WKUP:
	WKUP！！！！！！PA0
  LED2 lights means exit standby.
  LED3 lights means  standby mode exit by WKUP Pin's Rising edge.
