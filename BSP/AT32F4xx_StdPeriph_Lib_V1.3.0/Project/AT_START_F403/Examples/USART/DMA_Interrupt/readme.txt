@par Example Description 

  This demo provides a basic communication between USART2 and USART3 using DMA 
  capability, flags and interrupts.If received data right ,the Three leds will 
  turn on.
  
  - Connect USART2 Tx pin (PD.05) to USART3 Rx pin (PC.11)
  - Connect USART2 Rx pin (PD.06) to USART3 Tx pin (PC.10)
