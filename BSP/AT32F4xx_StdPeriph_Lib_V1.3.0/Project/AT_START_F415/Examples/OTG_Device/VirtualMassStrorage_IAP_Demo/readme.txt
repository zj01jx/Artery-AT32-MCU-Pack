@par Example Description 

The Mass Storage IAP demo example gives a typical example of how to use the AT32F415
USB OTG Device peripheral to communicate with a PC Host using the bulk
transfer while the microSD card is used as storage media.	

More details about this Demo implementation is given in the User manual "AT32 USB MSD IAP应用指南_V1.0.1.pdf"
